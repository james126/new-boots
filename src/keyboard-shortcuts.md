# ⚡ Common VS Code Shortcuts

| Action | Windows/Linux | Mac |
|--------|----------------|-----|
| Command Palette | Ctrl+Shift+P | ⌘+Shift+P |
| Format Code | Shift+Alt+F | ⌥+Shift+F |
| Toggle Terminal | Ctrl+` | ⌘+` |
| Go to Definition | F12 | F12 |
| Rename Symbol | F2 | F2 |
| Quick Fix | Ctrl+. | ⌘+. |
| Find All References | Shift+F12 | Shift+F12 |
| Comment Line | Ctrl+/ | ⌘+/ |
